## [Unreleased]

### Added
- Initial release of Bookmark Manager
- Web-based bookmark organization with categories
- Support for icons and abbreviations
- CRUD operations (Create, Read, Update, Delete) for bookmarks
- YAML-based storage system
- Responsive web interface
- Environment variable configuration for bookmark file path
- Default bookmark categories (Developer, Social, Entertainment, Shopping)

### Features
- Flask web framework backend
- Clean, modern UI with gradient background
- Category-based organization
- Configurable bookmark file location
- Auto-creation of bookmark file with defaults if not present
- Error handling and validation
