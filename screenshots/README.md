# Screenshots

This directory contains screenshots for the Homepage Bookmark Manager documentation.

## Required Screenshots

To complete the documentation, please add the following screenshots:

1. **main-interface.png** - Screenshot of the main bookmark manager interface showing:
   - Categorized bookmarks
   - Icons and descriptions
   - Add bookmark button
   - Clean, responsive layout

2. **add-bookmark.png** - Screenshot of the add bookmark modal showing:
   - Name and URL fields
   - Category dropdown with existing options
   - Icon upload area
   - Save/Cancel buttons

3. **category-dropdown.png** - Screenshot showing:
   - Category dropdown opened
   - List of existing categories
   - Option to type new category

4. **icon-upload.png** - Screenshot of icon management showing:
   - Icon upload interface
   - Visual preview of uploaded icons
   - Icon selection for bookmarks

5. **config-editor.png** - Screenshot of the configuration editor showing:
   - YAML editor interface
   - Syntax highlighting
   - Validation feedback
   - File selection (bookmarks.yaml, services.yaml, etc.)

6. **homepage-integration.png** - Screenshot showing:
   - Homepage dashboard with bookmarks
   - Demonstration of integration
   - Bookmarks appearing in Homepage

## Screenshot Guidelines

- Use a consistent browser and screen size
- Ensure good contrast and readability
- Include realistic bookmark data
- Show the application in action
- Use PNG format for best quality
- Recommended size: 1200x800px or similar 16:10 aspect ratio

## Taking Screenshots

1. Start the application: `python3 app.py`
2. Open browser to `http://localhost:5000`
3. Add sample bookmarks and data
4. Take screenshots of each feature
5. Save in this directory with the exact filenames listed above

The README.md file is already configured to reference these screenshot files.
